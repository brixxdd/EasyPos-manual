Introducción a EasyPOS

EasyPOS es un sistema integral de punto de venta diseñado para simplificar y optimizar la gestión de tu negocio. Con una interfaz intuitiva y capacidades avanzadas de análisis, te permite controlar inventarios, procesar ventas y obtener insights valiosos sobre el rendimiento de tu empresa.

¿Qué encontrarás en este manual?

  • • Guías paso a paso para cada función
  • • Capturas de pantalla detalladas
  • • Mejores prácticas recomendadas
  • • Solución de problemas comunes

Para Usuarios

Aprende a usar todas las funciones de la aplicación móvil, desde la gestión básica hasta las características avanzadas de análisis.

Para Administradores

Configura el sistema, gestiona usuarios, establece permisos y aprovecha las herramientas de análisis para la toma de decisiones.

Instalación y Configuración Inicial

1

EasyPOS está disponible para dispositivos Android e iOS. Visita tu tienda de aplicaciones correspondiente y busca "EasyPOS" o usa los siguientes enlaces:

Descargar en el

Disponible en
Google Play
2

Crea tu Cuenta

Al abrir la aplicación por primera vez, necesitarás crear una cuenta. Proporciona la siguiente información:

  • Nombre de tu negocio
  • Tu nombre completo
  • Correo electrónico válido
  • Contraseña segura (mínimo 8 caracteres)
  • Tipo de negocio

Importante

Guarda tu contraseña en un lugar seguro. EasyPOS no almacena contraseñas en texto plano por seguridad, por lo que no podemos recuperarla si la olvidas.

3

Configura tu Negocio

Después de crear tu cuenta, el asistente de configuración te guiará para establecer:

Información Básica

  • • Moneda principal
  • • Zona horaria
  • • Impuestos locales
  • • Información de contacto

Preferencias

  • • Idioma de la interfaz
  • • Formato de fecha/hora
  • • Notificaciones
  • • Métodos de pago

Gestión de Productos

Agregar Nuevos Productos

1

Accede a la sección Productos

Desde el menú principal, selecciona "Productos" y luego presiona el botón "+" en la esquina superior derecha.

2

Completa la Información del Producto

Completa todos los campos obligatorios:

Información Básica
  • • Nombre del producto
  • • Código SKU
  • • Categoría
  • • Descripción
Precios e Inventario
  • • Precio de venta
  • • Precio de costo
  • • Stock inicial
  • • Stock mínimo
3

Guarda el Producto

Revisa toda la información y presiona "Guardar". El producto quedará disponible para ventas inmediatamente.

Consejo

Usa códigos SKU únicos y descriptivos. Esto facilitará la búsqueda y el seguimiento del inventario a largo plazo.

Métodos de Pago

EasyPOS soporta diversos métodos de pago para adaptarse a las necesidades de tu negocio:

Efectivo

Pago en efectivo tradicional. El sistema calcula el cambio automáticamente.

Tarjeta

Débito o crédito. Requiere terminal POS externa para procesamiento.

Transferencia

Pago por transferencia bancaria o aplicaciones móviles.

QR/Digital

Pagos mediante código QR o billeteras digitales.

Solución de Problemas

Problemas Comunes

No puedo sincronizar los datos

Solución:

  1. Verifica tu conexión a internet
  2. Reinicia la aplicación
  3. Ve a Configuración → Sincronización y presiona "Sincronizar ahora"
  4. Si el problema persiste, contacta soporte técnico

El escáner QR no funciona

Solución:

  1. Asegúrate de que la cámara esté limpia
  2. Concede permisos de cámara a la aplicación
  3. Verifica que haya suficiente luz
  4. Mantén el código QR a 15-20 cm de distancia

No puedo imprimir recibos

Solución:

  1. Verifica la conexión de la impresora
  2. Asegúrate de tener papel térmico
  3. Reinicia la impresora y el dispositivo
  4. Verifica la configuración de la impresora en la app
