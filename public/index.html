Manual de Usuario Documentación Técnica

Características Principales

Todo lo que necesitas para gestionar tu punto de venta

Aplicación Móvil

React Native con Expo para Android e iOS. Operación offline completa con sincronización automática.

  • Interfaz intuitiva y responsive
  • Funcionamiento sin conexión
  • Escaneo QR integrado

Analytics Avanzado

Sistema completo de análisis y predicciones para optimizar tu negocio.

  • Predicciones de ventas
  • Alertas de stock automáticas
  • Dashboards interactivos

Seguridad Integral

Autenticación JWT, control de roles y permisos, encriptación de datos sensibles.

  • Autenticación JWT
  • Control de acceso RBAC
  • Encriptación bcrypt

Base de Datos Robusta

PostgreSQL con transacciones ACID, integridad referencial y respaldo automático.

  • Transacciones ACID
  • Integridad referencial
  • Respaldos automáticos

Integración WhatsApp

Comunicación automatizada con clientes y proveedores vía WhatsApp Business API.

  • Notificaciones automáticas
  • Envío de recibos
  • Alertas de stock

API RESTful

Backend escalable con Node.js/Express, endpoints bien documentados y versionado.

  • Endpoints RESTful
  • Documentación OpenAPI
  • Versionado de API
