Autenticación

POST /api/auth/login
Autenticar usuario

Request

{
  "email": "usuario@example.com",
  "password": "contraseña123"
}

Response (200)

{
  "success": true,
  "data": {
    "user": {
      "id": "uuid",
      "email": "usuario@example.com",
      "name": "John Doe",
      "role": "admin"
    },
    "tokens": {
      "access": "eyJhbGciOiJIUzI1NiIs...",
      "refresh": "eyJhbGciOiJIUzI1NiIs..."
    }
  }
}

Request

{
  "email": "nuevo@example.com",
  "password": "contraseña123",
  "name": "Jane Doe",

{

  "data": {
    "user": {
      "id": "uuid",
      "email": "nuevo@example.com",
      "name": "Jane Doe",
      "role": "admin"
    },
    "business": {
      "id": "uuid",
      "name": "Mi Negocio"
    }
  }
}

Productos

GET /api/products
Obtener lista de productos

Query Parameters

  • page (integer): Número de página (default: 1)
  • limit (integer): Productos por página (default: 20)
  • search (string): Búsqueda por nombre o SKU
  • category (string): Filtrar por categoría
  • inStock (boolean): Solo productos con stock

Response (200)

{
  "success": true,
  "data": {
    "products": [
      {
        "id": "uuid",
        "sku": "PROD-001",
        "name": "Producto Ejemplo",
        "price": 29.99,
        "stock": 150,
        "category": "Electrónicos"
      }
    ],
    "pagination": {
      "page": 1,
      "limit": 20,
      "total": 150,
      "pages": 8
    }
  }
}

Headers Requeridos

  • Authorization: Bearer <token>
  • Content-Type: application/json
POST /api/products
Crear nuevo producto

Request

{
  "sku": "PROD-002",
  "name": "Nuevo Producto",
  "description": "Descripción del producto",
  "price": 49.99,
  "cost": 25.00,
  "stock": 100,
  "minStock": 10,
  "categoryId": "uuid",
  "barcode": "1234567890123"
}

Response (201)

{
  "success": true,
  "data": {
    "product": {
      "id": "uuid",
      "sku": "PROD-002",
      "name": "Nuevo Producto",
      "price": 49.99,
      "stock": 100,
      "createdAt": "2025-01-15T10:30:00Z"
    }
  }
}

Ventas

GET /api/sales
Obtener lista de ventas

Query Parameters

  • startDate (string): Fecha inicial (ISO 8601)
  • endDate (string): Fecha final (ISO 8601)
  • cashier (string): ID del cajero
  • paymentMethod (string): Método de pago
  • status (string): Estado de la venta

Response (200)

  "success": true,
  "data": {
    "sales": [
      {
        "id": "uuid",
        "total": 89.97,
        "tax": 14.40,
        "subtotal": 75.57,
        "paymentMethod": "cash",
        "status": "completed",
        "cashier": {
          "id": "uuid",
          "name": "John Doe"
        },
        "items": [
          {
            "productId": "uuid",
            "productName": "Producto 1",
            "quantity": 3,
            "unitPrice": 29.99,
            "total": 89.97
          }
        ],
        "createdAt": "2025-01-15T15:30:00Z"
      }
    ],
    "summary": {
      "totalSales": 1247.50,
      "totalTransactions": 42,
      "averageTicket": 29.70
    }
  }
}

Manejo de Errores

Códigos de Estado Comunes

200

OK

Solicitud exitosa

201

Created

Recurso creado exitosamente

400

Bad Request

Solicitud inválida

401

Unauthorized

No autorizado

Formato de Error

{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Los datos proporcionados no son válidos",
    "details": [
      {
        "field": "email",
        "message": "El formato del email no es válido"
      },
      {
        "field": "password",
        "message": "La contraseña debe tener al menos 8 caracteres"
      }
    ]
  }
}

Códigos de Error Comunes

  • VALIDATION_ERROR: Error de validación de datos
  • NOT_FOUND: Recurso no encontrado
  • UNAUTHORIZED: Sin autorización
  • FORBIDDEN: Acceso prohibido
  • DUPLICATE_RESOURCE: Recurso duplicado
  • INSUFFICIENT_STOCK: Stock insuficiente

SDK y Librerías

Node.js SDK 

const EasyPOS = require('easypos-sdk');

const client = new EasyPOS({
  apiKey: 'your-api-key',
  baseURL: 'https://api.easypos.com'
});

// Obtener productos
const products = await client.products.list({
  page: 1,
  limit: 20
});

// Crear venta
const sale = await client.sales.create({
  items: [
    { productId: 'uuid', quantity: 2 }
  ],
  paymentMethod: 'cash'
});
npm install easypos-sdk

Python SDK 

from easypos import EasyPOSClient

client = EasyPOSClient(
    api_key='your-api-key',
    base_url='https://api.easypos.com'
)

# Obtener productos
products = client.products.list(
    page=1,
    limit=20
)

# Crear venta
sale = client.sales.create({
    'items': [
        {'product_id': 'uuid', 'quantity': 2}
    ],
    'payment_method': 'cash'
})
pip install easypos-python
